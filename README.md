# Personal Portfolio

A clean, responsive portfolio website to showcase your work and skills.

## 🚀 Features

- **Responsive Design**: Looks great on all devices
- **Modern UI**: Clean and professional design
- **Easy to Customize**: Update content and styling with minimal effort
- **Fast & Lightweight**: Optimized for performance

## 🛠️ Technologies Used

- HTML5
- CSS3 (with CSS Variables)
- Vanilla JavaScript
- [Font Awesome](https://fontawesome.com/) for icons

## 📁 Project Structure

```
personal-portfolio/
├── index.html          # Home page
├── css/
│   └── styles.css      # Main stylesheet
├── js/
│   └── main.js         # JavaScript functionality
├── images/             # Store all images here
└── pages/              # Additional pages
    ├── about.html
    ├── projects.html
    └── contact.html
```

## 🚀 Getting Started

1. **Clone the repository**
   ```bash
   git clone https://github.com/yourusername/personal-portfolio.git
   ```

2. **Navigate to the project directory**
   ```bash
   cd personal-portfolio
   ```

3. **Open in your browser**
   - Simply open `index.html` in your preferred web browser
   - Or use a local development server if you have one set up

## 🎨 Customization

1. Update `index.html` with your personal information
2. Add your projects to `pages/projects.html`
3. Customize colors and fonts in `css/styles.css`
4. Add your own images to the `images/` directory
